# 🧠 MeetingMind AI

AI-powered meeting insight extractor built with **Streamlit**.

This application analyzes meeting transcripts and automatically extracts:

* Decisions
* Action items
* Deadlines
* Open issues
* Responsibility mapping
* Intelligence score

---

# 📂 Supported File Types

* PDF
* DOC
* DOCX
* TXT

---

# ⚙️ Installation

Clone the repository:

```
git clone https://github.com/SelenaJathanna/MeetingMind-AI.git
cd MeetingMind-AI
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the app:

```
streamlit run app.py
```

The application will open in your browser.
